Programming experience:

Informatica, middelbaar,
ervaring, game dev

Op het middelbaar heb ik 3 jaar lang informatica gevolgd, 2 jaar op het VWO en 1 jaar op de HAVO. Heel veel ervaring heb ik hier niet gekregen maar wel een basis voor codeertaal structuren en talen, voornamelijk HTML, CSS, Javascript, Python en SQL. Zelf heb ik in mijn eigen tijd een paar jaar terug een eigen game gemaakt op Unity en met Pygame, deze games waren heel simpel maar hebben mij wel veel laten leren over programmeertalen en hebben me laten leren dat ik game development ook heel leuk vind.
